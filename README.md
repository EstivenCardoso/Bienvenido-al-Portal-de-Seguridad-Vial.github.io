# Portal de Seguridad Vial y Criminalística

## Descripción
Este es un portal web educativo dedicado a la promoción de la seguridad vial y la divulgación del trabajo de la criminalística en la investigación de accidentes de tránsito. El sitio incluye información sobre señales de tránsito, estadísticas de accidentes, consejos de prevención, un blog sobre criminalística vial, y un cuestionario interactivo para evaluar conocimientos.

## Características
- **Secciones principales**: Inicio, Señales de Tránsito, Estadísticas, Prevención, Blog Criminalística, Cuestionario.
- **Galería de imágenes**: Muestra imágenes relacionadas con la seguridad vial y criminalística.
- **Gráficos interactivos**: Estadísticas visuales usando Chart.js.
- **Blog con búsqueda y comentarios**: Artículos sobre criminalística vial con funcionalidad de búsqueda y comentarios almacenados localmente.
- **Cuestionario educativo**: 10 preguntas para evaluar conocimientos sobre seguridad vial.
- **Diseño responsivo**: Adaptable a diferentes dispositivos.

## Cómo usar
1. Abre el archivo `index.html` en tu navegador web preferido.
2. Navega por las secciones usando los botones del menú de navegación.
3. Interactúa con los elementos: ve videos, responde el cuestionario, busca en el blog, etc.

## Tecnologías utilizadas
- **HTML5**: Estructura del sitio.
- **CSS3**: Estilos y diseño responsivo.
- **JavaScript**: Funcionalidad interactiva, gráficos con Chart.js, carrusel de imágenes, blog y cuestionario.
- **Chart.js**: Librería para gráficos estadísticos.

## Requisitos
- Un navegador web moderno (Chrome, Firefox, Edge, etc.) que soporte HTML5, CSS3 y JavaScript.
